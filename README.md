# Vedantic Mappings

Verified Cartographies of the Non-Dual Path

This project contains comprehensive, single-page HTML cartographies documenting the teachings of two great sages of the Advaita Vedanta tradition: Ramana Maharshi and Nisargadatta Maharaj. Each cartography provides a complete map from mistaken identity to liberation, grounded in scriptural sources and presented with scholarly precision.

## 🌐 Usage

### Opening the Cartographies

1. **Start Here**: Open `index.html` in any modern web browser
2. **Navigate**: Click on either cartography card to explore
3. **Direct Access**: You can also open individual HTML files directly:
   - `ramana-maharshi.html` — Ramana Maharshi's teaching
   - `nisargadatta-maharaj.html` — Nisargadatta Maharaj's teaching

### Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Theme Toggle**: Light/dark mode support with system preference detection
- **Smooth Navigation**: Animated sections and scroll-linked navigation
- **Interactive Elements**: Tooltips on Sanskrit terms, navigation tooltips, and click-to-expand koshas

## 📚 Cartographies Overview

### I. Ramana Maharshi — The Sage of Arunachala

**Teaching**: Self-Enquiry (Atma Vichara)

**Lineage**: Advaita Vedanta tradition of Adi Shankaracharya

**Key Features**:
- The 1896 death experience that became liberation
- Seven stages of realization from mistaken identity to the Self
- Detailed mapping of Self-enquiry as experiential practice
- Comprehensive Upanishadic sources and Mahavakyas
- Integration of jnana, bhakti, and karma yoga paths
- Interactive glossary with click-to-expand koshas section
- Elegant tooltip system for Sanskrit terms
- Enhanced navigation with improved sidebar functionality

**Primary Sources Documented**:
- Nan Yar? (Who Am I?, 1902)
- Ulladu Narpadu (Reality in Forty Verses, 1928)
- Upadesa Saram (Essence of Instruction, 1927)
- Talks with Sri Ramana Maharshi (1935–39)
- Be As You Are (ed. David Godman, 1985)

**Scriptural Foundations**:
- Mandukya Upanishad (four states of consciousness)
- Brihadaranyaka Upanishad (neti-neti method)
- Chandogya Upanishad (Tat Tvam Asi)
- Kena Upanishad (witness behind knowing)
- Bhagavad Gita
- Shankara's Vivekachudamani

---

### II. Nisargadatta Maharaj — The Inchegiri Sampradaya

**Teaching**: Nisarga Yoga (Yoga of the Natural State)

**Lineage**: Navnath lineage via Sri Siddharameshwar Maharaj

**Key Features**:
- Radical distinction between consciousness and pure awareness
- Seven stages from vyakti (person) to Parabrahman (Absolute)
- Detailed analysis of the "I Am" as Ishwara/God
- Complete mapping of the mechanism of Maya
- Integration of love as the nature of the Absolute
- Interactive glossary with detailed Vedantic connections
- Enhanced navigation with mobile responsiveness
- Elegant quote design matching Ramana cartography styling
- Substantive content replacing excessive quotes

**Primary Sources Documented**:
- I Am That (trans. Maurice Frydman, 1973)
- Prior to Consciousness
- Seeds of Consciousness
- The Core of His Teaching (Pradeep Apte)
- Conversations compiled by Nic Higham

**Scriptural Foundations**:
- Mandukya Upanishad (states of consciousness)
- Inchegiri Sampradaya teachings
- Navnath lineage wisdom
- Direct pointing to the Absolute

## 🔧 Technical Implementation

### File Structure
- Each cartography is a single, self-contained HTML file
- CSS is embedded in `<style>` tags within the `<head>`
- JavaScript is embedded in `<script>` tags before closing `</body>`
- No build process or external dependencies required

### Performance
- Minimal external requests (only Google Fonts)
- Optimized CSS with CSS variables for theming
- Smooth animations using CSS transforms
- Intersection Observer for scroll-triggered animations
- Efficient JavaScript for interactive elements

### Accessibility
- Semantic HTML structure
- ARIA labels for interactive elements
- Keyboard navigation support
- Sufficient color contrast ratios
- Responsive text sizing with clamp()
- Tooltips and click interactions with proper states

## 📝 Notes on Approach

### Scholarly Precision
- All teachings are grounded in primary sources
- Sanskrit terms are provided with scriptural references
- Lineage connections are explicitly documented
- Cross-tradition parallels are noted where relevant
- External links and resources are verified for accuracy

### Experiential Focus
- Emphasis is on direct experience over philosophical argument
- Practices are presented as verifiable methods, not beliefs
- The maps point to what can be discovered, not what must be believed
- Liberation is presented as recognition, not attainment

### Non-Sectarian Presentation
- Both cartographies honor their respective lineages
- No attempt is made to harmonize differences artificially
- Each teaching is presented on its own terms
- Students are encouraged to study both for comparative understanding
